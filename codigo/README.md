### conversor de medidas.
-   unidades aceitas:
       > celsius
       
       > fahrenheit

### Como utilizar 

```go
go run conversor.go {Valor_para_conversão} {Valor_para_conversão} {Valor_para_conversão} {UNIDADE}
```
### exemplo
```go
go run conversor.go 32 27.4 -3 0 celsius
```
ou 

```go
 go run conversor.go 32 27.4 -3 0 quilometros
```
